About

This projects aims to be a centralized hub for data revelant to HAM radio operators.
The data consists of publicly available data and is provided freely (as in "free of charge" as well as in "freedom") but without any warranty.
The idea behind and design of this site are highly inspired by eddb.io.

Help wanted

The aim of this project is global. Collecting good data for things around the globe is not an easy thing to do.
If you have access to relevant data or know someone who does please get in contact.
If want to contribute, thank you! If you have the time to do so there are multiple ways to do so:

  • Report issues of this site
  • Report issues of the API behind it
  • Report incorrect data
  • Directly contribute to the code of this site
  • Directly contribute to the API behind it
  • Get in contact to support data collection and preparation
  • Help translating this site
  • Help translating translatable data
